Bahasa Indonesia

    PENTING! (yang belum tau cara memasangnya). Sebelum kalian memainkan game ini, kalian harus mengunduh peluncurannya terlebih dahulu sebelum kalian mengunduh fangame atau mod ini. perlu mengunduh peluncuran CYF (create your frisk) agar kalian bisa memasang mod tersebut.

Cara pemasangan
  1. unduh CYF (CreateYourFrisk)
  2. extrak file ke folder yang ingin anda letakan
  3. Download tundra Geno
  4. pindahkan file Tundra Geno ke folder CYF lalu masukan ke folder mods
  5. buka CYF
  6. lalu tekan enter / klik kanan (mouse)
  7. pilih mod Tundra Geno

English

    urgent! (who don't know how to install it). Before you play this game, you have to download the launch before you download this game or mod. need to download launch CYF (create your frisk) so you can install the mod.

How To Install
  1. Download CYF (CreateYourFrisk)
  2. extract the file to the folder that you want to put
  3. Download Tundra Geno
  4. move the Tundra Geno file to the CYF folder then enter the mods folder
  5. Open CYF
  6. then press enter / right click (mouse)
  7. Select the Tundra Geno mod
